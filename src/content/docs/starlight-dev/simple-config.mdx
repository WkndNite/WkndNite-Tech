正如之前提及到的，现在就可以在项目里找到 docs 文件夹，向其中添加 md 或者 mdx 格式的文件，然后就可以在侧边栏中看到这些文件了。写好文件之后，我们还需要签到根目录下的 astro.config.mjs 文件中，添加一些配置。

对于其中的starlight实例可以向sidebar里添加配置来设置侧边栏目录

而当前可以注意到右目录显示的并不是汉语，而是默认的英文，这是因为astro默认使用的是英文，因此我们需要在astro.config.mjs文件中添加一些配置来设置语言。
