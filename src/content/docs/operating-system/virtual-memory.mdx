## 背景介绍

---

虚拟内存是内存管理的一种技术，它允许执行进程时不必完全载入内存，可以部分程序载入到内存。优点在于逻辑地址空间可以大于物理地址空间，可以被多个进程共享地址空间，可以提供更有效的进程创建。

## 按需调页

---

可以减少 I/O 操作，可以减少内存使用，应答时间快。

> Q : 如何区分哪些页要在物理内存，哪些要在虚拟内存？

> A : 根据有效/无效位。
